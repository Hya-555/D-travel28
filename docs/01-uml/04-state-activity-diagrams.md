# 状态图与活动图 — 旅游业务管理系统

## 1. 申请状态图

```mermaid
stateDiagram-v2
    [*] --> 草稿: 开始办理申请
    草稿 --> 已付订金: 支付订金完成
    已付订金 --> 已录入参加者: 录入全部参加者
    已录入参加者 --> 已完成: 确认申请完成
    草稿 --> 已取消: 取消申请(申请完成前)
    已付订金 --> 已取消: 取消申请(申请完成前)
    已录入参加者 --> 已取消: 取消申请
    已完成 --> 已取消: 取消申请(扣除手续费)

    已完成 --> 余款已付: 支付余款完成

    note right of 已完成
        完成后可办理:
        - 参加者变更/取消
        - 整个申请取消
        - 新增参加者视为新申请
    end note
```

## 3. 申请办理活动图

```mermaid
graph TD
    A([开始]) --> B[查询旅游团]
    B --> C{截止日期检查}
    C -->|已过期| D[提示不可申请]
    C -->|未过期| E{人数限额检查}
    E -->|已满| D
    E -->|有空位| F[录入责任人姓名和电话]
    F --> G[录入大人和小孩人数]
    G --> H[计算订金]
    H --> I{距出发天数}
    I -->|≥60天| J[订金 = 总价×10%]
    I -->|30~59天| K[订金 = 总价×20%]
    I -->|<30天| L[订金 = 总价(全款)]
    J --> M[录入订金支付]
    K --> M
    L --> M
    M --> N[打印收据和申请书]
    N --> O[交付给顾客]
    O --> P([结束])
    D --> P
```

## 4. 取消申请活动图

```mermaid
graph TD
    A([开始]) --> B[查询申请]
    B --> C{取消类型}
    C -->|整个申请| D[计算取消手续费]
    C -->|部分参加者| E[选择要取消的参加者]
    E --> F{被取消者是责任人?}
    F -->|是| G[选定新责任人]
    F -->|否| D
    G --> D
    D --> H{距出发天数}
    H -->|>30天| I[手续费=0]
    H -->|10~30天| J[手续费=已付×20%]
    H -->|1~9天| K[手续费=已付×50%]
    H -->|0天| L[手续费=已付×100%]
    I --> M[退款=已付-手续费]
    J --> M
    K --> M
    L --> M
    M --> N[保存取消记录]
    N --> O[退还退款给顾客]
    O --> P([结束])
```
